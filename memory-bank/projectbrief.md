# Project Brief: Farm Follies

## Overview
**Farm Follies** is a physics-based stacking arcade game where players catch barnyard animals being flung from a tornado, stack them carefully, and bank them before the tower topples.

**Tagline:** "Monday morning. Tornado warning. Total chaos."

## Core Requirements

### Gameplay
1. Tornado moves across top of screen, dropping animals
2. Player controls a farmer who catches animals
3. Caught animals stack on farmer's head
4. Stack wobbles based on physics - can collapse
5. Bank animals to score points before collapse
6. Special animal variants have unique abilities

### Animals (9 Types)
- Chicken, Duck, Goose (light)
- Pig, Goat, Sheep (medium)
- Cow, Rooster, Horse (heavy)
- Each has normal + special variants with abilities

### Key Mechanics
- Wobble physics with collapse risk
- Combo scoring system
- Power-ups for assistance
- Progressive difficulty
- Bush bounce system (from cow poop ability)

## Target Platforms
- Web browser (desktop + mobile)
- Touch and mouse controls
- Responsive design

## Visual Style
- Procedural 2D graphics (canvas)
- Rustic farm color palette
- Animated tornado with debris
- Expressive animal faces

## Success Criteria
1. Smooth, responsive gameplay at 60fps
2. Satisfying physics feel
3. Clear visual feedback
4. Replayable with progression
5. Mobile-friendly controls
