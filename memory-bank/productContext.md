# Product Context: Farm Follies

## Why This Exists
Farm Follies is a casual arcade game designed to be:
- Immediately understandable
- Easy to play, hard to master
- Satisfying with physical feedback
- Replayable with variety

## Problems It Solves
1. **Boredom** - Quick, engaging gameplay sessions
2. **Skill expression** - Mastery through wobble management
3. **Variety** - Different animals and abilities keep it fresh

## User Experience Goals

### First-Time Player
- Understand controls in seconds (drag to move)
- Catch first animal successfully
- Experience the wobble physics
- Learn banking through UI prompts

### Returning Player
- Chase high scores
- Discover special animal variants
- Master wobble management
- Unlock achievements

### Core Loop
```
Catch Animals → Stack Grows → Wobble Increases → Bank or Risk → Repeat
```

## Emotional Journey
1. **Chaos** - Tornado drops animals unpredictably
2. **Tension** - Stack wobbles dangerously
3. **Relief** - Successfully bank stack
4. **Excitement** - Special abilities trigger
5. **Satisfaction** - High score achieved

## Key Differentiators
- Physics-based stacking (not just matching)
- Procedural animal graphics with personality
- Tornado as spawn mechanism (not just random drops)
- Bush bounce mechanic for second chances

## Game Modes
1. **Endless** - Default mode, play until game over
2. **Time Attack** - Maximum score in time limit
3. **Zen** - Relaxed, no lives
4. **Boss Rush** - Face powerful boss animals
